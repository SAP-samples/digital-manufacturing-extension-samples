    Tutorial: Line Monitor Shift Progress Plugin Extensions

    Line Monitor Shift Progress Plugin Extensions

    Introduction

    The Line Monitor Shift Progress Plugin implements four extensions that can be utilized by custom developers to modify the core plugins behavior:

    • LifecycleExtension
    • NotificationExtension
    • PluginEventExtension
    • PluginPropertyEditorExtension

    Provided Sample Extension for the Line Monitor Shift Progress Plugin

    This sample extension demonstrates how to:

    • Filter the list of resources which can be selected in the plugin

    PluginEventExtension

    The PluginEventExtension provides the following functions that can be implemented by a custom extension:

    • onFilterResources
    • onGetInitialResource
    • onCardDataRetrieved

    The following public functions are available to be called by extensions:

    • getCard
    • getCardFilters
    • formatNumericCardValues

    The sap.dm.dme.lmplugins.shiftProductionCardPlugin.controller.extensions.PluginEventExtensionConstants object defines the member functions used by the extension, as well as useful model and property names.
