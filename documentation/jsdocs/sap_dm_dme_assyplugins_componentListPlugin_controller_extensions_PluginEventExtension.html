    sap/dm/dme/assyplugins/componentListPlugin/controller/extensions/PluginEventExtension

    sap.dm.dme.assyplugins.componentListPlugin.controller.extensions.PluginEventExtension provides functions that handles plugin events. Custom extensions can be created to override or modify core behaviour.

    Constructor

    new sap/dm/dme/assyplugins/componentListPlugin/controller/extensions/PluginEventExtension()

    Description:

    • Constructor for the Component List Plugin Event Extension. The following POD Plugin event functions can be overridden by custom extensions:

         onWorklistSelectEvent()
   onInputChangeEvent()
   onOperationChangeEvent()
   onPodSelectionChangeEvent()
   onAssemblyPointComponentChangeEvent()
   onComponentAssembledEvent()
   onProductionProcessSuccessEvent()
   onAutoAssembledSfcStateChangedEvent()
   onComponentListRefreshEvent()
   loadComponents()
   onAssembleButtonPressedEvent()
   onRemoveButtonPressedEvent()
   onErrorRemovingComponent()
   createAndRefreshTable()
   updateListConfiguration()
   updateTableConfiguration()
   updateColumnConfiguration()

      The following public functions are available to extensions: 

         refreshComponentList()
   getPluginContainer()
   getComponentListTable()
